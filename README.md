# umbrella_bending_machine


## [ 공유 우산 사업을 위한 자동 대여 단말기 ]

### 🚙 프로젝트 소개
------
TU_VCC에서 공유 우산 사업의 시제품을 위한
우산 자동 대여 단말기 개발

------
### 🕰️ 개발 기간
* 22.10 ~ 22.12


### ⚙️ 개발환경
* micropython
* **IDE** : Pycharm
* **Tool** : kicad, fusion360

어플리케이션과 MQTT를 통해 통신하며, 대여 정보에 따라 잠금장치 개방.
